**Política de Privacidad** Esta aplicación no recopila, almacena ni comparte información personal directamente por parte del desarrollador. La aplicación muestra anuncios mediante servicios de terceros, como Google AdMob. Estos servicios pueden recopilar información automáticamente para mostrar publicidad. Los datos que pueden recopilar incluyen: * Identificadores del dispositivo (como el ID de publicidad) * Dirección IP * Información básica sobre el uso de la aplicación * Ubicación aproximada Esta información es utilizada exclusivamente por los proveedores de anuncios para mostrar publicidad relevante. Para más información sobre cómo Google gestiona estos datos, puedes consultar su política de privacidad: https://policies.google.com/privacy El usuario puede desactivar la personalización de anuncios desde la configuración de su dispositivo móvil. **Cambios a esta política** Esta política puede actualizarse ocasionalmente. Se recomienda revisarla periódicamente. **Contacto** Si tienes preguntas sobre esta política, puedes contactarnos.
